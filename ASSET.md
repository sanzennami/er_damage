# ASSET

本目录缓存 Eternal Return 角色小图标，用于配合 `https://raw.githubusercontent.com/pypy-vrc/er-gamedata/master/data/Character.json` 的 `code` / `name` 字段离线使用。

## 来源

- 数据索引：`pypy-vrc/er-gamedata` 的 `data/Character.json`。
- 图标来源：Eternal Return Wiki / Fandom 的 `Category:Character thumbnails` 中各角色 `Mini.png` 文件。
- 文件命名：`assets/characters/{code三位}-{name}.png`。
- 机器可读清单：`assets/characters/manifest.json`。

## 特殊命名映射

| er-gamedata name | Fandom 文件名基准 |
| --- | --- |
| `DebiMarlene` | `Debi & Marlene Mini.png` |
| `Istvan` | `István Mini.png` |
| `LiDailin` | `Li Dailin Mini.png` |
| `Lyanh` | `Ly Anh Mini.png` |
